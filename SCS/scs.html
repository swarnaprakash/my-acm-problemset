Problem Name: SCS

Time Limit : 2s

A supersequence of a string is a new string which is formed from the original string by inserting some of the characters without disturbing the relative positions of the original characters.For example "ABCDEFGH" is a supersequence of "AGH",but is not a supersequence of "AHG".

The shortest common supersequence of two strings (SCS) is a common supersequence of minimal length. For two input strings, an SCS can be formed from an LCS(longest Common subsequence) easily.By inserting the non-LCS symbols from both strings into LCS ,while preserving the symbol order, we get the SCS.

Given two strings your task is to find the length of the SCS.

Input:

Number of testcases t &le 100 on the first line.Each test contains two strings in separate lines.The strings will contain ONLY uppercase alphabets.Length of the strings &le 100.

Output:

Output one line per test containing the length of the SCS.

Sample Input:

3
ABC
BAC
AGH
ABCDEFGH
AHG
ABCDEFGH

Sample Output:

4
8
9
