sandy has bought 'n' saplings and he wants to plant them in his garden.He wishes to plant them in rows(where the number of rows is atleast 2) and all rows should have the same number of saplings(where the number of saplings in each row is atleast 2).Given n your task is to find how many ways sandy can plant them.

Input:

On the first line is the number of testcases t<=50.Then t lines follow each containing a single integer n<=2147483647 (n will fit in a 32-bit signed int).

Output:

one line per test,containing the number of ways

Sample Input:

2
2
6

Sample Output:

0
2
