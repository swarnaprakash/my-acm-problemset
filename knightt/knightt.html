Problem Name : Knight's Trip

Time Limit : 3s

The Knight's Tour is a famous mathematical problem involving a knight on a chessboard. The knight is placed on the empty board and, moving according to the rules of chess, must visit each square exactly once.

We are now interested in a simpler problem Knight's Trip.In this problem the knight should make a trip from any given square in a chessboard to another given square in the same board.This must be done in minimum number of moves.The knight cannot move out of the chess board at any time.

The chessboard squares are labelled with Algebraic Chess Notation.

Input:

Input begins with number of testcases t &le 1000.Each test has two squares labelled with Algebraic Chess Notation.

Output:

For each test output an integer representing the minimum moves required for a knight to move from one square to another.

Sample Input:

3
a1 h8
a1 b1
c3 d5

Sample Output:

6
3
1
