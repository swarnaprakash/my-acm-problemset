Easy Sorting

Time Limit: 3s

Given a list of student records you have to sort them.Each student record contains name , age ,marks in 5 subjects. Sort them in ascending order of total marks.In case ,two students have the same total marks the one that appears earlier in the input should appear before the other one in the sorted list.

Input:

Number of testcases t &le 200 on the first line.Each test begins with the number of records n &le 100 .Each record is represented in three separate lines.First line has the name of the student,which will be a string of ONLY lowercase letters of atmost 20 characters. Next line has an integer representing the age.Next line has 5 space separated integers each &le 100 representing the marks in 5 subjects.

Output:

Output the sorted list of records.Each record should in the same format as in the input.Print a blank line after each test case.

Sample Input:

1
5
sivaguru
20
100 90 90 90 85
swarnaprakash
20
100 100 100 100 100
santhanakrishnan
20
50 50 50 50 50
vijay
20
50 50 50 50 50
sunilkumar
20
90 90 90 95 90

Sample Output:

santhanakrishnan
20
50 50 50 50 50
vijay
20
50 50 50 50 50
sivaguru
20
100 90 90 90 85
sunilkumar
20
90 90 90 95 90
swarnaprakash
20
